# Cost of Living Project

## Project Description
The Cost of Living Project is an application that compares the cost of living in different cities and provides users with a visual representation of how costs vary depending on the geographical area.

## Features
- **Cost Comparison**: Compare living costs in various cities.
- **Data Visualization**: Interactive charts and graphs to visualize cost comparisons.
- **Search Functionality**: Easily search for a city to view its cost of living data.
- **User-Friendly Interface**: Simple and intuitive UI for better user experience.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (React)
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Data Visualization**: Chart.js, D3.js

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SuhasGowda24/Mini_Project-Cost_of_living.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Mini_Project-Cost_of_living
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Open your browser and navigate to `http://localhost:3000` to access the application.
- Use the search bar to select the cities you want to compare.
- Explore various visualizations for a detailed understanding of living costs.
